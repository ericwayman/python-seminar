Included are img_gui.py and default.jpg

img_gui.py
Creates a GUI to to allow for an image search via google images.  A query is entered into the search bar.  The program will return the first image returned by a google images search.  The gui also features 3 buttons for image manipulation.  An image can be rotated 90 degrees clockwise, flipped about the y-axis or made into a gray-scale image.

default.jpg should be in the same directory as img_gui.py.  It is the default image that displays in the image window of img_gui.py.
